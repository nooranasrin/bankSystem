Features : 
--------------
1. creating an account in bank
2. deposit
3. Withdrawal
4. money tranfer (account to account)
5. bank statement
6. balance enquiry

Datas:
---------
1. creating an account in bank

  input
  -------
  -> name, dob, address, phone, aadhar_no, select_bank, select_branch

  output
  ------
  -> account_number
  -> pin


2. deposit

  input
  -------
  -> name, account_number, amount

  output
  ------
  ->status of deposit


3. Withdrawal

  input
  -------
  -> account_number, pin,  amoun

  output
  ------
  ->status of Withdrawal


4. Transfer

  input
  -------
  -> account_number, pin, beneficiry_account_number, ifsc code, amount

  output
  ------
  ->status of Transfer

5. bank statement

  input
  -------
  -> account_number, pin

  output
  ------
  ->last 10 transactions


6. Balance enquiry

  input
  -------
  -> account_number, pin

  output
  ------
  ->balance
